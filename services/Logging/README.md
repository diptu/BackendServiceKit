# Logging Service

## Responsibility

Stores every log generated by every service.

### Examples

- User logged in
- Policy denied
- Tenant created
- Payment failed
- Database timeout
- Redis disconnected
- Email sent
- JWT expired

Everything is searchable.

Logs are immutable.

---

## Typical APIs

| Method | Endpoint |
|----------|-----------|
| POST | `/logs` |
| POST | `/logs/bulk` |
| GET | `/logs` |
| GET | `/logs/{id}` |
| DELETE | `/logs/{id}` |
| GET | `/logs/search` |
| GET | `/logs/tenant/{tenant_id}` |
| GET | `/logs/service/{service}` |
| GET | `/logs/user/{user_id}` |
| POST | `/logs/export` |
